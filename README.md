# 📊 Employee Data Analysis

## 📌 Project Overview

This project contains a structured **MySQL SQL script** developed to perform data management, and analysis on an Employee Database.

The script answers multiple business and analytical questions using SQL queries, helping derive insights related to employees, departments, and salary structures.



---

## 🗂️ Dataset Structure

The database is built using the following datasets:

* **Employee Details** — Basic employee information
* **Department Details** — Department mapping and structure
* **Salary Details** — Compensation and salary records

These datasets are imported into MySQL tables and used for relational analysis.

---

## 🛠️ Tools & Technologies Used

* **MySQL**
* **CSV Data Import**

---

## 🧩 Key SQL Operations Performed

The script includes queries related to:

* Database and table creation
* Data insertion and imports
* SELECT queries
* Filtering using WHERE
* Sorting using ORDER BY
* Aggregations (SUM, AVG, COUNT)
* GROUP BY analysis
* JOIN operations
* Subqueries
* Salary and department insights
